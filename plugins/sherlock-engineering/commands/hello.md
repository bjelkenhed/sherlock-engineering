# hello

A simple hello world command to demonstrate how Claude Code commands work.

## Usage

Invoke this command by typing `/sherlock-engineering:hello` in Claude Code.

## Behavior

When invoked, this command should:

1. Greet the user warmly
2. Explain that this is a starter command from the sherlock-engineering plugin
3. Provide a brief overview of what the plugin can do
4. Suggest next steps for customizing the plugin

## Example Output

```
Hello! Welcome to Sherlock Engineering.

This is a starter plugin for Claude Code. You can customize it by:
- Adding new commands in the `commands/` directory
- Creating skills in the `skills/` directory
- Defining agents in the `agents/` directory

Run /help to see all available commands.
```
